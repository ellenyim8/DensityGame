# Game
Created by Ellen Y.

Density 
Left side: 
Controls: **A = left, W = up, S = down, D = right**
Right side player: 
Controls: **arrow buttons (up, left, right, down)**

Pills points lowest (1) to highest (4) 
- Black = 4
- Red = 3
- Blue = 2
- Yellow = 1

Goal: whoever reaches 4000 points wins

Modules: 
- pycharm
- random
- os (operating system) 
- sys (output sys)
- time
- math
