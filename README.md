# Game
Density 

2 player game

left side player: 
Controls: A = left, W = up, S = down, D = right

right side player: 
Controls: arrow buttons (up, left, right, down) 

Pills points: 
- Black = 4
- Red = 3
- Blue = 2
- Yellow = 1

Goal: whoever reaches 4000 points wins

Modules: 
- pycharm
- random
- os (operating system) 
- sys (output sys)
- time
- math
